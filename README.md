# Warum diese Guides wichtig sind

Viele Wasserprobleme im Whirlpool entstehen durch ein Zusammenspiel aus mehreren Faktoren:

- Wasserwerte
- organische Belastung
- Filtration
- Nutzung

Die Guides in diesem Repository helfen dabei, typische Probleme zu verstehen und dauerhaft zu vermeiden.
